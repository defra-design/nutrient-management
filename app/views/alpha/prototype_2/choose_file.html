Import data from the Planet desktop software

What you can import

You can only import data from a a valid Planet v3.3 farm file.

It can take up to 5 minutes to import a file, depending on it's size

How to import a Planet v3.3 farm file

  • 1. Select the 'Choose file' button
  • 2. Find and select your Planet farm file.
  • By default your Planet file will be saved under 'Documents', 'Planet', and then 'Data'. It will have the name you have given it and a '.mdb' file extension.
  • 3. Select 'Upload file'.
