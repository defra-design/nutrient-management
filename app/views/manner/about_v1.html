How estimating nutrient supply from organic materials works using MANNER-NPK

MANNER-NPK (Manure Nitrogen Evaluation Routine) estimates the nutrient supply from applications of organic materials.

This lets you:

  • compare different organic material applications to see how nutrient supply could be affected (scenario planning)
  • see the financial value of these applications - how much you would save compared to buying inorganic fertiliser

The MANNER-NPK software will estimate:

  • total nitrogen (N)
  • nitrogen (N) losses
  • crop available nitrogen (N)
  • total and crop available phosphate (P2O5), potash (K2O), sulphur (SO3) and magnesium (MgO)

What you'll need

You will need to enter a few simple inputs to use MANNER-NPK:

  • farm and field details (you may already have these set up in NMPT-GB)
  • manure type and nutrient content (laboratory analysis results or default nutrient content)
  • manure application details (including application rate, application method and soil incorporation)
  • Economic value of organic materials

You can calculate the potential financial value of your manure application(s) based on fertiliser prices. You will be able to input your prices for inorganic fertilisers, allowing you to directly compare them with the nutrient value of your organic materials.

This comparison will help you understand the cost savings you could achieve by using manure instead of buying inorganic fertilisers.

Saving and printing

You can save your estimates as a PDF and print them if you wish.
