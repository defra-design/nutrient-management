Release schedule for the plan and manage nutrient applications tool

Last updated x November 2025

Public Beta release, December 2025

The first functionality in the nutrient management planning tool is ready for use by the public in England and Wales. This has had approximately eight months of user testing.

The first public release helps users to;

  • create a nutrient management plan for arable crops and grassland
  • get nutrient recommendations from RB209
  • plan organic and inorganic material applications, which include MANNER - NPK calculations
  • see warning messages where applications may breach Nitrate Vulnerable Zone (NVZ) rules
  • update plans and record what happens at a field level

This version also helps to create NVZ reports for:

  • N-max
  • Livestock manure nitrogen (N) farm limit

Next releases, early 2026

Launch in Scotland

The Scottish version of the nutrient management planning tool is currently in development. This will be released to all users in early 2026, after user testing. This version will give recommendations based on the Scottish Technical Notes. It will help users to create NVZ reports based on Scottish rules.

Next available functionality

Upcoming releases available in England, Scotland and Wales will include the ability to:

  • create an NVZ report for existing manure storage capacity
  • calculate NVZ organic manure storage requirements and manure inventory
  • use MANNER NPK to create quick estimates of nitrogen (N) value and losses
  • import data from existing PLANET files

The current functionality will continue to be updated based on feedback from our users.
