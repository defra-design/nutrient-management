About this service

Getting started

The first step is to set up your farm and add fields. Fields then have crops and nutrients added to them. If you have more than one holding, it's best to add a separate farm for each one.

Soil analysis

You will get better recommendations if you add recent soil analysis for each field. You can update these as you get new analysis results.

Nitrate Vulnerable Zone (NVZ) rules

The service will warn you of possible breaches of the NVZ rules, if your field is within an NVZ.

Soil Nitrogen Supply (SNS)

The service can calculate SNS based on field history and previous crop. Alternatively, you can add SNS analysis results at the end of the planning process.

Importing PLANET data

If you used PLANET to make nutrient plans before you can import this data to set up a farm.
