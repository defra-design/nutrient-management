Get help or give feedback

This service is currently in a private beta phase. This means that we are testing the service with a group of invited users. As such, some functionality may not work as expected and some features have not yet been developed.

Check the release schedule to see what features have been added to the test version of the service.

View the terms of use for this service.

Getting help

If you experience a technical problem with any of the functionality in this service, you can email: helpdesk@rsk-bsl.co.uk.

The helpdesk will help with issues such as:

  • technical issues
  • login issues
  • service outages

Please tell us what you were trying to do, what happened, and what you expected. This will help us to identify the problem.

Help with the service's recommendations

You can email the helpdesk if you have any issues with the data and recommendations the service provides. Email: helpdesk@rsk-bsl.co.uk.

This service might give slightly different recommendations from the previous service PLANET. This is because this new service uses an updated version of the RB209 API, so it provides the very latest data.

We will reply to your email as quickly as we can.

Giving feedback about this service

If you would like to give us feedback about this service, then please email: nmpt@defra.gov.uk.

Your feedback will help us improve this service.

More information

The Nutrient Management Plan is intended to be used alongside RB209

For advice about nutrient planning in general, start with Tried and Tested (opens in a new window).

For more information on RB209, visit Nutrient management guide (RB209) section 1 (opens in a new window).
