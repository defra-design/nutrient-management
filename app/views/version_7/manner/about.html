How estimating nutrient supply from organic materials (MANNER NPK) works

This tool uses MANNER NPK to estimate the nutrient supply from organic materials.

First select a field, then select the crop type you want base an estimate on.

Add the type of organic material you are applying as well as the amount and application method.

The tool will estimate:

  • the total nitrogen (N)
  • crop available nitrogen (N)
  • nitrogen (N) losses
  • Phosphate (P2O5), Potash (K2O), Sulphur (SO3) and Magnesium (MgO) values

You can add more applications to the same estimate and change any of the application details. For example, you can compare different organic material types, or application rates.

Enter your inorganic fertiliser prices to calculate the value of your organic materials.

You can print your estimates or download them as a PDF.
